# About the Exam Venue Data

This repository contains data about exam venues scheduled between January 26, 2026 and February 7, 2026.
This is open data for public use.

## Data Details

- Period: January 22, 2026 – February 7, 2026
- Data is separated by region (e.g., eastern_japan/, western_japan/)
- Each CSV file contains information about exam venues

## Usage

This data is intended to be imported and used with Google Maps My Maps. You can upload the CSV files to My Maps to visualize the exam venues on a map.

### How to Import CSV Data into Google Maps My Maps

1. Go to [Google My Maps](https://www.google.com/maps/d/).
2. Click on "Create a new map".
3. In the new map, click on "Import" under the untitled layer.
4. Select the CSV file you want to upload (e.g., from the `eastern_japan/` or `western_japan/` folder).
5. Follow the prompts to choose the columns for the map location (e.g., address or latitude/longitude).
6. The exam venues will be displayed on the map.
7. You can customize the map as needed and save it to your Google account.
